# 装饰器

什么是装饰器呢？
举个例子，我们想要有一根过水的水管，现在我们想要监测这个水管的过水情况怎么办呢?
我们可以给这跟水管接上水表。







代码：
```py
import logging


def warning(func):
    '''警告'''
    def wrapper(*args, **kwargs):
        logging.warning("{} is running".format(func.__name__))
    return wrapper

def error(func):
    '''错误'''
    def wrapper(*args, **kwargs):
        try:
            return func(*args, **kwargs)
        except Exception as e:
            return HttpResponse(e)
    return wrapper
```
